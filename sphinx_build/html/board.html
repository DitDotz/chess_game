class board.Board#

Bases: object

Represents the chessboard.

board (Dict[Tuple[int, int], Piece]): A dictionary representing the positions of pieces on the board. king_in_checkmate (bool): Indicates whether the game is in a checkmate state. moves_made (int): The number of moves made in the game. expected_player (Color): The color of the player expected to make the next move.
